Game Instructions:

Try to guess the secret word! You'll recieve 'bulls' and 'cows' depending on how close your answer is to the secret word!

Bulls: Number of CORRECT letters in the CORRECT position

Cows: Number of CORRECT letters in the WRONG position

Press the "PLAY GAME" button to begin!

----------------------------------------------------------------------

Input heterogram to the database. (Heterogram: A word with no repeating letters)
